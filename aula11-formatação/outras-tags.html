Outras tags em HTML

Base

Essa tag é usada para especificar o URL

, base para todas as URLs em um documento HTML.

Menu

A tag costumava ser usada para definir menus no HTML,

mas saeu uso foi desencorajado em favor de outras abordagens
.

Font

A tag font era usada para definir o tamanho,a cor, e a face da fonte em um documento HTML,

mas seu uso não é mais recomendado devido a práticas de estilo mais modernas usando o CSS.

frame

Essas tags eram usadas para criar layouts de página baseados em frames,

mas agora são desencorajadas em favor de aboradagens mais modernas usando HTML5 e CSS.

frameset

Define um conjunto de frames organizado por múltiplas janelas.

Noframes

Texto exibido para navegadores que não lidam com "frames".

Acronym

Essa tag era usada para definir sigla ou uma abreviação em um documento HTML, mais seu uso foi desencorajado em favor de outras aboradagens, como o uso de elementos semânticos.

Dir

Essa tag era usada para para criar uma lista de diretorios em um documento HTML, mas seu uso foi desencorajado em favor de elementos semânticos, como UL e li

S

Essa tag era usada para representar texto que não é mais relevante ou preciso em um documento HTML, mais seu uso foi desencorajado em favor de outras abordagens, como o uso de CSS para estilizar o texto.

Basefont

Define as propriedades da font para todo o texto do documento

Center

Usado para alinhar texto e conteúdo no centro.

Ex: texto alinhado no centro

Strike

Exibe texto rasurado Ex: TEXTO RASURADO

tt

Define teletipo de texto.

Col

Define os atributos da coluna da tabela.

Colgroup

Define um grupo de colunas da tabela
